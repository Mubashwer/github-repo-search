To continue using the GitHub Browser Plugin, you need to authenticate with GitHub to increase your API rate limit from 60 to 5,000 requests per hour.

Step 1: Generate a Personal Access Token

Click the button below to open GitHub's token creation page:

Create Token on GitHub

Step 2: Configure Token Settings

On the GitHub page:

  • Set a descriptive name like "GitHub Browser Plugin"
  • Select these scopes:
    • repo (optional, for private repos)
    • read:user (for user information)
  • Click "Generate token"
